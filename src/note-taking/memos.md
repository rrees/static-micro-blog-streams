# Memos

A basic note-taking web application that allows the notes to be searched by text, date and hashtag. It can be run locally and uses SQLite and a folder so it is easy to backup notes with existing drive replication.

The server seems much more efficient to run than Electron but you do need to pin a tab or have some kind of PWA pin.
